### Installation
```
npm i
```
In the chrome dev tools settings, go to the Console section and tick "Enable custom formatters".

### Development
```
npm run dev
```

### Web Build
```
npm run build
```

### App Build
OSX:
```
npm run app:osx
```

Linux
```
npm run app:linux
```

### Gihtub pages deployment
```
npm run deploy
```

### Release app
```
npm version [major|minor|patch]
git push && git push --tag
```
